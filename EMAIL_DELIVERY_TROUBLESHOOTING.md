# 이메일 수신 문제 해결 가이드

## 현재 상황
- Brevo API를 통해 이메일 발송 성공
- 메시지 ID 확인됨: 실제 발송 완료
- 사용자가 이메일을 받지 못함

## 이메일 확인 체크리스트

### 1. Gmail 확인 사항
- **받은편지함**: 기본 받은편지함 확인
- **스팸 폴더**: 가장 일반적인 원인
- **프로모션 탭**: Gmail이 자동으로 분류하는 탭
- **소셜 탭**: 때로는 소셜 탭으로 분류됨

### 2. 검색으로 찾기
Gmail에서 검색어 입력:
- `from:support@amusefit.com`
- `AmuseFit 인증번호`
- `인증번호`

### 3. 필터 설정 확인
- Gmail 설정 → 필터 및 차단된 주소
- AmuseFit 관련 필터가 있는지 확인

### 4. 이메일 주소 오타 확인
- qdy061230@gmail.com 정확한지 재확인

## 대안 해결방법

### 즉시 사용 가능한 방법
